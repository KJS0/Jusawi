# 변경점
## [0.26.0] - 2025-10-27
### Added.
- 트랙패드 제스처: 화면에 꽉찬(또는 그 이하 배율) 상태에서 두 손가락 좌우 스와이프로 이전/다음 사진 이동.
- 대각선 팬: 두 손가락 스크롤로 가로/세로를 동시에 이동.
- Alt+휠 정밀 줌 지원, 줌 단계/정밀 단계 설정 추가.

### Fixed
- 전체화면에서 줌·팬 중 툴바/필름스트립이 가장자리에 딱 붙지 않거나 흔들리던 문제를 해결(즉시 숨김, 위치 고정, 불필요한 재배치 최소화).
- 전체화면 전환/해제 시 필름스트립과 평점 바의 표시 순서가 바뀌던 문제 해결.

### Removed
- 설정의 단축키 탭 제거(간소화).
- 보기 고정/공유 기능 및 관련 옵션 제거.

## [0.25.2] - 2025-10-27
### Added
- 전체화면 토스트: 평점/플래그 변경 시 화면 중앙 알림.

### Changed
- 보기/줌 안정성: 화면 맞춤/가로/세로 맞춤 재적용 일관성, 자유 줌 단계 가변화, 더블클릭(실제 크기↔화면 맞춤) 동작 일관성, 스크롤바 비활성화.

- 일부 환경에서 DPI에 따라 썸네일 정렬이 어긋나던 문제 수정.

## [0.21.0] - 2025-10-13
### Added
- 사진 정보 한눈에 보기: 상단 `정보` 버튼을 누르면 우측에 사진의 핵심 정보(촬영 일시, 파일/폴더, 용량, 해상도/화소수, ISO, 초점 거리, 노출 보정(EV), 조리개, 셔터속도, GPS)가 보기 좋게 표시됩니다.
- 주소 자동 표시: 사진에 위치 정보가 있으면 GPS 좌표 바로 아래에 실제 주소를 자동으로 보여줍니다(국내는 카카오, 해외는 구글 사용).
- 지도로 위치 보기: 정보 패널 하단에 지도가 함께 표시됩니다. 지도를 클릭하면 브라우저에서 구글 지도로 바로 열어 확인할 수 있습니다.


### Fixed
- 다양한 카메라의 EXIF를 더 잘 읽도록 호환성을 높였습니다.
- 사진을 바꿨을 때 정보가 가끔 갱신되지 않던 문제가 해결되었습니다.
- 지도를 클릭해도 사진 더블클릭 동작에 영향을 주지 않습니다.

## [0.18.0] - 2025-09-07
### Added
- 대용량 이미지 최적화(스케일 디코딩): 처음엔 화면에 맞춘 크기에 맞게 불러오고, 필요할 때 원본으로 자동 업그레이드됩니다.
- AI 분석: 사진 요약(짧은 캡션, 설명), 촬영 의도, 태그 자동 생성. EXIF는 보조 증거로만 사용하며 불확실 정보는 ‘추정’으로 표기.
- 자연어 검색(프리뷰): “노을 해변”, “도시 야경” 등 문장 질의로 유사 사진 검색. GPT 임베딩 기반, 상위 일부는 비전 모델로 재검증해 정확도 향상.

### Changed
- 100% 이하 배율에서도 스케일 캐시를 적극 활용하도록 조정해 처음 표시 속도가 더 빨라졌습니다.
- 좌표 갱신을 뷰포트 중심이 아닌 “현재 커서 위치” 기준으로 바꿔, 마우스를 대기한 상태에서도 X/Y 표기가 튀지 않도록 개선했습니다.
- 빠르게 사진을 넘길 때 디코더가 폭주하지 않도록 원본 업그레이드에 짧은 텀(디바운스)을 두었습니다.
- 검색 결과 UI를 윈도우 탐색기와 유사한 썸네일 그리드로 개선(16:9 고정 썸네일, 단일 클릭 미리보기, 더블클릭 열기).

### Fixed
- 썸네일 → 원본 전환 시 상태바 배율/좌표가 순간적으로 틀어지거나 깜빡이는 현상을 해결했습니다.
- 내비게이션(이전/다음)으로 바꾼 뒤 썸네일만 남아 보이던 문제를 고쳤습니다.
- 애니메이션(GIF/WebP)에서 다음 이미지로 이동할 때 이전 프레임이 잠깐 보이던 문제를 방지했습니다.
- 화면 맞춤(Fit) 상태에서 원본 업그레이드가 이루어지면 즉시 재맞춤되어 보이는 크기가 안정적으로 유지됩니다.
